### 一、系统总架构

```c
                ┌──────────────────────┐
                │   学生信息管理系统      │
                └──────────┬───────────┘
                           │
                 ┌─────────▼─────────┐
                 │      登录模块       │
                 │  注册 / 登录 / 校验 │
                 └─────────┬─────────┘
                           │
        ┌──────────────────┼──────────────────┐
        │                  │                  │
┌───────▼───────┐  ┌───────▼───────┐  ┌───────▼────────┐
│    学生端      │  │    教师端      │  │    管理员端      │
└───────┬───────┘  └───────┬───────┘  └───────┬────────┘
        │                  │                  │
  只读功能模块        学生管理模块         账号管理模块
  成绩查询            增删改查             用户管理
  成绩分析            排名分析             数据导出
                                        待办处理
```

### 二、系统运行流程

```c
程序启动
   │
   ▼
登录/注册界面
   │
   ▼
身份校验成功
   │
   ▼
根据角色分流
   ├── 学生端
   ├── 教师端
   └── 管理员端
   │
   ▼
执行对应功能
   │
   ▼
保存数据
   │
   ▼
程序结束
```
